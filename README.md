# AI Office Hours Helper

A small web app that converts free-form workflow pain points into structured action plans using an LLM.

## Features
- Free-text workflow input
- Python backend (Netlify Function)
- Real LLM call
- Strict schema validation
- Human-readable + raw JSON output
- Deployed on Netlify

## Tech Stack
- Vanilla HTML/CSS/JS
- Python Netlify Functions
- OpenAI API
- Pydantic
